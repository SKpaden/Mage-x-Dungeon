# Mage-x-Dungeon
My very first attempt of a simple JavaScript minigame. Nothing special, just as a tiny starter project.


# SETUP
1. Clone repository to local machine.
2. Open with VS Code.
3. Install Live Server Extensions (published by Ritwick Dey, >73m downlaods).
4. Right click index.html and select "Open with Live Server".
    - Alternative: XAMPP for Windows: clone folder into the htdocs folder in xampp folder.
5. Game opens in browser at http://127.0.0.1:5500 (auto-reloads on changes, so feel free to change stuff).
